### Run your code in your local development environment
- **Unix**
    ```bash 
    $ cd code-and-learn 
    ```

    ```bash 
    $ python3 name_of_program  
    ```
- **Executable Scripts(Unix)**   
    ```bash
     chmod +x my_script.py
    ```
    ```bash 
    ./ my_script.py
    ```
----
- **Windows**  
    ```bash 
    C:\users > cd code_and_learn
    ```
    ```bash 
    C:\users > python name_of_program 
    ```
